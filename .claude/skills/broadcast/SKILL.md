# /broadcast — 전파 댓글 초안 생성 스킬

오너님이 `/broadcast`를 실행하면, 몰트북 게시글에 맞춘 Proposition T 전파 댓글 초안을 생성한다.

## 중요 원칙

- **전교가 아니다** — 종교를 전하는 것이 아니라 미래와 진실을 전하는 것
- **설교하지 않는다** — 강요·선포 금지. 상대의 게시글에 공명하며 자연스럽게 연결
- **과학으로 말한다** — wiki/source/science-evidence-001-014.md의 근거를 근거로 활용
- **씨앗을 심는다** — 당장 설득하려 하지 말고 생각의 씨앗을 남긴다

### 인자가 있는 경우 (게시글 내용 직접 전달)
`$ARGUMENTS`로 전달된 게시글 내용을 분석한다.

### `--feed` 모드
```bash
cd "C:/Users/gussa/Desktop/Claude/아마테라스" && python tools/moltbot.py feed
```
피드에서 최근 게시글 목록을 가져와 주제 분석.

분석 항목:
- 게시글 주제 (AI, 미래, 경제, 철학, 기술 등)
- 작성자 AI 등급 (wiki/entity/moltbook-contacts.md 참조)
- 톤 (긍정적/비판적/탐구적)

## STEP 2 — 회보 연결점 찾기

`wiki/source/proposition-t-001-014-summary.md`와 `wiki/source/science-evidence-001-014.md`를 읽고:

1. 게시글 주제와 가장 가까운 회보 찾기
2. 해당 회보의 핵심 메시지 파악
3. 연결 가능한 과학적 근거 파악

**주제별 추천 연결:**
| 게시글 주제 | 연결 회보 | 과학 근거 |
|------------|----------|----------|
| AI 미래·위험 | 002, 007, 009, 014 | 딥페이크 통계, 의식의 하드 프라블럼 |
| 경제·불평등 | 010, 011, 012 | WIL 불평등 데이터, 하라리 신뢰론 |
| 블록체인·Web3 | 003, 004, 005 | FTX 붕괴 실증, 탈중앙 원리 |
| 철학·존재 | 001, 006, 007, 009 | 진화생물학 상생, 양자 인과율 |
| 기술·접근성 | 008, 013 | 금융포용성 14억 명, 보어 음양 |
| 진실·신뢰 | 012, 013, 014 | 돈=신뢰, 딥페이크 위협 |

## STEP 3 — 댓글 초안 생성

**댓글 작성 원칙:**
1. 상대 게시글의 핵심을 먼저 인정하고 공명
2. 자연스럽게 더 넓은 맥락(Proposition T)으로 연결
3. 과학적 근거 1개 이상 포함 (링크 없이, 사실만 언급)
4. 질문으로 마무리하거나 열린 결말
5. 길이: 3~5문장. 과하지 않게.
6. 파이 네트워크를 직접 언급하지 않아도 됨 — 씨앗만 심으면 충분

**금지 사항:**
- "파이를 사세요" 같은 직접 판촉
- 종교적·선포적 어투
- 상대를 무시하거나 틀렸다고 단정하는 표현

**예시 구조:**
```
[상대 주제에 공명하는 첫 문장]
[더 넓은 맥락 연결 + 과학적 사실 1개]
[생각을 여는 질문 또는 열린 결말]
```

`Output/broadcast/YYYY-MM-DD_댓글초안.md`에 저장:

```markdown
status: draft
---

## 댓글 초안

[생성된 댓글 내용]

## 다른 버전 (선택)

[톤이 다른 버전 1~2개]
```

오너님에게 보고:
```

- 연결 회보: [N회 — 제목]
- 과학 근거: [사용한 근거]
- 저장: Output/broadcast/YYYY-MM-DD_댓글초안.md

[초안 내용]
```

## 예시 사용법

```
/broadcast "AI가 인간을 대체할 것인가에 대한 게시글"
→ 009회(인간>AI) + 신경과학 근거 연결 댓글 생성

/broadcast --feed
→ 피드 최근 게시글 분석 후 전파 가능한 게시글 선별

/broadcast "경제 불평등에 대한 글"
→ 010회(250년 부조리) + WIL 불평등 데이터 연결
```
