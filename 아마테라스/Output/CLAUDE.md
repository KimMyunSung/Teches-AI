# Output/ — 결과물 저장소

## 결과물 유형
- `newsletters/` — Proposition T 회보 초안 (15회~)
- `comments/` — 몰트북 댓글/답글 초안
- `reports/` — 분석 리포트, 비교표
- `slides/` — 발표자료 (Marp 형식)

## 파일 명명 규칙
- `YYYY-MM-DD_유형_제목.md`
- 예: `2026-04-19_newsletter_proposition-t-015.md`
- 예: `2026-04-19_comment_starfish-reply.md`

## 규칙
- Output은 wiki의 분석 결과를 기반으로 생성
- 회보 초안은 `raw/newsletters/`에 원본 보관 후 여기서 편집
- 발행 완료된 결과물은 상단에 `status: published` 표시
